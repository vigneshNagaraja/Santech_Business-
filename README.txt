Fillable Weekly Timesheet — Corrected Version
============================================
- Per-diem: default $50/day, **editable per day**.
- No per-diem total row (since some days may be $0). TOTAL PAY = Rate×Hours + sum(per-diem per day).
- Per-day signature column (fillable text) + bottom signature **inputs** (name + date) so you can type.
- Mobile-friendly scaling (no cramped scrolling).

Deploy (GitHub Pages):
1) Upload index.html + styles.css to your repo root.
2) Enable Pages (Deploy from main, folder /(root)).
3) Share the Pages link with your team.

Use:
- Enter Date, Time In/Out (lunch mins default 30), Per Diem (50 or 0 on no-money days).
- Rate/hr optional; total pay auto-updates.
- Save keeps data on the device; Clear resets all.
- Print → Save as PDF for submission.
