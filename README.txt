Fillable Weekly Timesheet — Per Diem + Mobile Friendly
=====================================================
- Per-diem column (default $50 per day, editable)
- Auto-calc: daily hours, weekly hours, per-diem total, total pay (rate/hr + per diem)
- Signature lines formatted: "Employee ____ Date ____" and "Approval ____ Date ____"
- Better mobile scaling (single-column header on phones, tighter table)

Deploy:
1) Upload index.html + styles.css to your repo root.
2) Enable GitHub Pages (Deploy from main, folder /(root)).
3) Share the Pages link.

Use:
- Enter time (24h or 12h depending on device). Lunch mins default 30.
- Per-diem defaults to $50/day. Change if needed per day.
- Rate/hr optional; total pay = (rate × total hours) + total per diem.
- Save stores data on the device (localStorage). Clear resets.
