
Blank Weekly Timesheet (Print-Ready)
====================================

Files:
- index.html  (single-page printable form)
- styles.css  (print styles)

How to publish for your team:
1) Upload index.html and styles.css to your GitHub repo root.
2) Enable GitHub Pages (Deploy from branch: main, Folder: /(root)).
3) Share the Pages link on WhatsApp.

How to print:
- Open the page in any browser → Ctrl+P / Cmd+P → Save as PDF (A4 or Letter).
- Mobile: Open link → Share → Print → Save as PDF.
